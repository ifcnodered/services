<h2> Project </h2>
<p>this project demonstrates services to <p>
  <ul>
    <li>convert IFC-SPF to JSON</li>
    <li>xml based validation using schematron</li>
    <li>using linked-data algorithms to view the IFC </li>
</ul>
<br>
